# ist-task

- Ruby 2.7.1
- Rails 6.1
- PostgreSQL

## Getting Started

To get up and running:

- Clone the repo to your local machine:

SSH

git clone git@github.com:team-infinikorn/ist-task.git

HTTPS

git clone https://github.com/team-infinikorn/ist-task.git

- Install dependencies

bundle install
gem install foreman

- Prepare DB

rails db:create
rails db:migrate
